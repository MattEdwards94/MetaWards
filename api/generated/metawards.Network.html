metawards.Network¶

class metawards.Network(nodes: metawards._nodes.Nodes = None, to_links: metawards._links.Links = None, play: metawards._links.Links = None, weekend: metawards._links.Links = None, nnodes: int = 0, nlinks: int = 0, plinks: int = 0, max_nodes: int = 10050, max_links: int = 2414000, to_seed: List[int] = None, params: metawards._parameters.Parameters = None)¶

This class represents a network of wards. The network comprises nodes (representing wards), connected with links which represent work (predictable) links. There are also additional links for play (unpredictable/random) and weekend
