metawards.WardID¶

class metawards.WardID(home: Union[metawards._wardid.WardID, str, int, None] = None, commute: Union[metawards._wardid.WardID, str, int, None] = None)[source]¶

A simple class that identifies a Ward (or Wards), or a connection between Wards. This could be used, e.g. to specify ward moves
